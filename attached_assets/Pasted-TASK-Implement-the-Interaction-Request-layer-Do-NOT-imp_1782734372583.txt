CONTEXT

Every interaction in the game should begin as an InteractionRequest.

An InteractionRequest is a raw interaction produced by Minecraft or another game system.

Examples:

- Player sent chat
- Player right-clicked NPC
- Player attacked NPC
- Player offered trade
- Another NPC started a conversation
- Settlement issued an order

The InteractionRequest is intentionally raw.

It contains only what happened.

Later, an InteractionResolver will translate it into an InteractionContext.
